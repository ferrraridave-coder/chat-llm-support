Common Questions

Do I need an account?
You can use Chat LLM with your own OpenRouter or OpenAI API keys. Using these services may require an account with those providers.

Is there a subscription?
No. Chat LLM offers a 14-day trial and a simple one-time purchase for permanent access.

What models are supported?
Chat LLM supports a wide range of OpenRouter text and vision models. Some model types that output images or video are not currently supported.

How do I report a bug?
Send an email with a description of the issue and, if possible, a screenshot. You will receive a response as soon as possible.
