Frequently Asked Questions

What are Chat LLM and LLM Pro?

Chat LLM is a privacy-focused AI chat app for iPhone and iPad with multiple AI options: built-in local model, cloud inference, real-time voice conversation, or bring your own API key for access to 300+ additional models.

LLM Pro is a premium AI chat app featuring 8 curated high-performance models with real-time web search on 4 models, transparent AI reasoning visualization (reasoning models only), voice conversation, and optional bring-your-own-key access to 300+ cloud models.

Do I need an account or API key?

Chat LLM:
No account or API key required to get started. The app includes:
- Built-in Local Model: Llama 3.2 1B runs completely offline on your device
- Cloud Inference: Fast and free cloud-based AI (Llama 3.1 8B), without an API key
- Optional BYOK: Add your own OpenRouter, OpenAI, or xAI API keys for 300+ models and voice chat

LLM Pro:
No account or API key required to get started. The app includes:
- Cloud Inference: Free tier includes fast and free cloud-based AI (Llama 3.1 8B), without an API key.
- Premium subscription (£9.99/month) unlocks 8 fast curated models with 4 featuring real-time web search. - Optional BYOK: Add your own OpenRouter, OpenAI, or xAI API keys for 300+ models and voice chat

What models are supported?

Chat LLM:
- Built-in Local Model: Llama 3.2 1B runs locally on your device with complete privacy. Text-only.
- Cloud Inference: Llama 3.1 8B provides fast cloud responses without requiring an API key. Text-only.
- Voice Chat: Real-time voice conversation powered by Grok with 5 voices (requires xAI API key).
- Optional Cloud Models (BYOK): 300+ OpenRouter and OpenAI models with your own API keys.

LLM Pro:
- Free Cloud Model: Fast cloud-based AI available without subscription
- Premium Subscription: 8 curated models including Llama 3.3 70B, Llama 3.1 8B, Llama 4 Scout, Llama 4 Maverick, Qwen 3 32B, Kimi K2, GPT-OSS 20B, and GPT-OSS 120B
- Web Search: Real-time web search available on Llama 4 Scout, Llama 4 Maverick, Qwen 3 32B, and Kimi K2
- Thinking Box: Transparent visualization of AI reasoning and research process (reasoning models only).
- Voice Chat: Real-time voice conversation powered by Grok with 5 voices (requires xAI API key).
- Optional Cloud Models (BYOK): 300+ OpenRouter and OpenAI models with your own API keys

What is Voice Chat?

Voice Chat provides real-time voice conversation with AI powered by Grok's speech API. Available in both Chat LLM and LLM Pro. You can choose from 5 different voices and enjoy unlimited continuous responses without time limits. Voice conversations are automatically transcribed and saved for reference. Voice Chat requires an xAI API key from x.ai.

How is my data handled?

Chat LLM - Local Model: When using the built-in local model, all conversations remain on your device. No data is transmitted anywhere, and the app works completely offline.

Both Apps - Cloud Inference: When using free cloud inference options, your messages are sent to our backend server which forwards them to the AI provider. We do not store your conversations.

LLM Pro - Premium Subscription: When using premium subscription models, your messages are processed through our Groq backend. We do not store your conversations.

Both Apps - Voice Chat: When using voice chat, your voice input and conversations are sent directly to xAI (Grok). Neither app stores your voice data or transcripts on our servers.

Both Apps - Cloud Models (BYOK): When using cloud models with your own API keys, your prompts, images, and documents are sent directly to the AI provider you selected (OpenRouter, OpenAI, or xAI). Neither app stores your prompts or uploads on our servers. For details on how cloud providers handle data, please refer to their privacy policies.

Can I use the app offline?

Chat LLM: Yes, the built-in local model works completely offline with no internet connection required. Cloud inference, voice chat, and cloud models with API keys require an internet connection.

LLM Pro: No, all features require an internet connection as the app operates exclusively via cloud-based APIs.

Why can't I upload images or documents?

Image uploads are only available when using vision-capable models in either app. The free cloud model (Llama 3.1 8B) and voice chat support text-only conversations. Chat LLM's local model is also text-only.

What is the LLM Pro subscription?

LLM Pro offers a premium subscription at £9.99 per month, providing access to 8 curated high-performance AI models. Four models include real-time web search capabilities. When using reasoning-capable models (available through subscription or via OpenRouter/OpenAI BYOK), the thinking box displays the AI's reasoning and research process in real-time. Subscription is managed through Apple's App Store and can be cancelled at any time from your device settings.

What devices are supported?

Both Chat LLM and LLM Pro are universal apps optimized for iPhone and iPad running iOS 18.0 or later.
