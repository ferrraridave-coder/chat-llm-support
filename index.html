Frequently Asked Questions

What are Chat LLM and LLM Pro?

Chat LLM is a privacy-focused AI chat app for iPhone and iPad with multiple AI options: a built-in local model, free cloud inference, and optional bring-your-own API keys for real-time voice conversation and access to 300+ additional models.

LLM Pro is a premium AI chat app for iPhone and iPad with multiple AI options: free cloud inference, 8 curated high-performance models with real-time web search on 4 models (with subscription), and optional bring-your-own API keys for real-time voice conversation and access to 300+ additional models.

Do I need an account or API key?

Chat LLM:
No account or API key required to get started. The app includes:
- Built-in Local Model: Llama 3.2 1B runs completely offline on your device.
- Free Cloud Inference: Llama 3.1 8B, text-only, without an API key.
- Optional BYOK: Add your own OpenRouter, OpenAI, or xAI API keys for 300+ models and real-time voice chat.

LLM Pro:
No account or API key required to get started. The app includes:
- Free Cloud Inference: Llama 3.1 8B, text-only, without an API key.
- Premium subscription (£9.99/month): Unlocks 8 curated models with 4 featuring real-time web search.
- Optional BYOK: Add your own OpenRouter, OpenAI, or xAI API keys for 300+ models and real-time voice chat.

What models are supported?

Chat LLM:
- Built-in Local Model: Llama 3.2 1B, runs locally, text-only.
- Free Cloud Inference: Llama 3.1 8B, text-only, no API key required.
- Optional Voice Chat: Real-time voice conversation powered by Grok (requires xAI API key).
- Optional Cloud Models (BYOK): 300+ OpenRouter and OpenAI models.

LLM Pro:
- Free Cloud Inference: Llama 3.1 8B, text-only, no API key required.
- Premium Subscription: 8 curated models including Llama 3.3 70B, Llama 3.1 8B, Llama 4 Scout, Llama 4 Maverick, Qwen 3 32B, Kimi K2, GPT-OSS 20B, and GPT-OSS 120B.
- Web Search: Available on Llama 4 Scout, Llama 4 Maverick, Qwen 3 32B, and Kimi K2.
- Thinking Box: Displays AI reasoning and research process (reasoning models only).
- Optional Voice Chat: Real-time voice conversation powered by Grok (requires xAI API key).
- Optional Cloud Models (BYOK): 300+ OpenRouter and OpenAI models.

What is real-time Voice Chat?

Real-time Voice Chat provides real-time voice conversation with AI powered by Grok's speech API. Available in both Chat LLM and LLM Pro. Conversations are transcribed and saved locally. Voice Chat requires an xAI API key from x.ai.

How is my data handled?

Chat LLM – Local Model: All conversations remain on your device and work offline.

Both Apps – Free Cloud Inference: Messages are sent to our backend and forwarded to the AI provider. We do not store conversations.

LLM Pro – Premium Subscription: Messages are processed through our Groq backend. We do not store conversations.

Both Apps – Voice Chat: Voice data is sent directly to xAI (Grok). No voice data or transcripts are stored on our servers.

Both Apps – Cloud Models (BYOK): Content is sent directly to the selected AI provider. We do not store prompts, images, or documents.

Can I use the app offline?

Chat LLM: Yes, when using the built-in local model only.

LLM Pro: No, all features require an internet connection.

Why can't I upload images or documents?

Image and document uploads are only available when using vision-capable models. Free cloud inference and voice chat are text-only.

What is the LLM Pro subscription?

LLM Pro offers a £9.99/month subscription providing access to 8 curated models, with web search on 4 models.

What devices are supported?

Both apps support iPhone and iPad running iOS 18.0 or later.
