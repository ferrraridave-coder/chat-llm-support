Frequently Asked Questions

What are Chat LLM and LLM Pro?

Chat LLM is a privacy-focused AI chat app for iPhone and iPad. It supports a free local on-device AI model, a free cloud AI model, optional bring-your-own API keys, and advanced features such as real-time voice chat with live web search, image and document uploads, and access to hundreds of external AI models.

LLM Pro is a premium AI chat and image generation app for iPhone and iPad. It combines high-performance cloud AI models, real-time web search, vision capabilities, and built-in AI image generation with clear usage limits.

Do I need an account or API key?

Chat LLM:
No account or API key is required to get started. The app includes a free on-device AI model that works completely offline, and free cloud AI access powered by Groq. You may optionally add your own OpenRouter, OpenAI, or Gemini API keys to access additional models, real-time voice chat, and web search.

LLM Pro:
No account or API key is required to get started. The app includes free cloud AI access powered by Groq. A premium subscription unlocks additional cloud models powered by Groq and Cerebras, as well as AI image generation. Optional bring-your-own API keys are supported for advanced users.

What models are supported?

Chat LLM:
- Free on-device AI model (Llama 3.2, runs locally — no internet required)
- Free cloud AI model powered by Groq
- Optional bring-your-own API keys for access to 300+ third-party models via OpenRouter or OpenAI
- Optional real-time voice chat powered by Gemini (requires a Gemini API key)

LLM Pro:
- Free cloud AI model powered by Groq
- Premium subscription unlocks curated high-performance cloud AI models powered by Groq and Cerebras
- Vision-capable models for image analysis
- Optional bring-your-own API keys for access to additional third-party models

What is AI image generation in LLM Pro?

LLM Pro includes built-in AI image generation and image-to-image editing. Image generation and editing are processed via Replicate. The premium subscription includes up to 100 images per month. Image usage is clearly displayed in-app.

What is real-time web search?

Chat LLM supports two forms of real-time web search:

OpenRouter Web Search (text chat):
When enabled in settings, OpenRouter performs a live web search before the selected model responds and feeds the results into its context. This works with any OpenRouter model and is useful for current events, news, prices, or anything beyond the model's training data. A small per-search fee (~$0.004) is charged to your OpenRouter account on top of the normal model cost.

Gemini Voice Chat Web Search:
Real-time voice chat powered by Gemini includes built-in Google Search grounding at no extra cost. Gemini automatically searches the web when you ask about current events, weather, sports results, or other live information, and weaves the results naturally into its spoken reply.

LLM Pro:
Real-time web search is available on supported premium models and requires an internet connection.

What is real-time Voice Chat?

Chat LLM:
Real-time Voice Chat enables natural spoken conversations with AI powered by Google Gemini. This feature requires your own Gemini API key. Spoken input and conversation content are processed by Google. Voice chat includes built-in Google Search grounding, so Gemini can answer questions about current events and live information during your conversation. You can also attach photos or camera images mid-conversation for Gemini to discuss.

How is my data handled?

On-Device AI (Local Llama model):
When using the local Llama model in Chat LLM, all processing happens entirely on your device. No messages, data, or personal information are sent to any server.

Free Cloud AI (Groq):
Messages and conversation content are sent to Groq to generate AI responses. Conversations are not stored on our servers.

LLM Pro – Premium Cloud Models (Groq & Cerebras):
Depending on the model selected, messages and conversation content are sent to Groq or Cerebras to generate AI responses. We do not store conversations, images, or documents on our servers.

Bring Your Own API Key (OpenRouter, OpenAI):
When you use your own API key, messages and content are sent directly to the selected provider, such as OpenRouter or OpenAI. OpenRouter may route requests to different model providers depending on the model selected. When OpenRouter web search is enabled, search queries are also processed by OpenRouter. API keys are stored only on your device.

Voice Chat (Google Gemini):
When using real-time voice chat, spoken input, conversation content, and any images you attach are sent to Google to generate AI responses. Google Search grounding may be used automatically to answer questions requiring live information. No voice data or transcripts are stored on our servers. Your Gemini API key is stored only on your device.

AI Image Generation (Replicate):
When generating or editing images, prompt text and any selected images are sent to Replicate for processing. Replicate processes this data to generate AI images within the app. Images are not stored on our servers.

Can I use the app offline?

Chat LLM: The free on-device Llama model works fully offline with no internet connection required. All cloud-based features — including Groq, OpenRouter, OpenAI, and Gemini voice chat — require an internet connection.

LLM Pro: Requires an internet connection for all features.

Why can't I upload images or documents?

Image and document uploads are only available when using models that support vision or document input. Feature availability depends on the selected model.

What is the LLM Pro subscription?

LLM Pro offers a £9.99/month subscription that unlocks premium cloud AI models powered by Groq and Cerebras, real-time web search on supported models, and AI image generation processed via Replicate with a monthly limit.

What devices are supported?

Both apps support iPhone and iPad running iOS 18.0 or later.
