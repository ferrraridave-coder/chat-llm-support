Frequently Asked Questions

What are Chat LLM and LLM Pro?

Chat LLM is a privacy-focused AI chat app for iPhone and iPad. It supports free cloud AI, optional bring-your-own API keys, and advanced features such as real-time voice chat and access to a wide range of external AI models.

LLM Pro is a premium AI chat and image generation app for iPhone and iPad. It combines high-performance cloud AI models, real-time web search, vision capabilities, and built-in AI image generation with clear usage limits.

Do I need an account or API key?

Chat LLM:
No account or API key is required to get started. The app includes free cloud AI access. You may optionally add your own OpenRouter, OpenAI, or xAI API keys to access additional models and real-time voice chat.

LLM Pro:
No account or API key is required to get started. The app includes free cloud AI access. A premium subscription unlocks additional cloud models and AI image generation. Optional bring-your-own API keys are supported for advanced users.

What models are supported?

Chat LLM:
- Free cloud AI model available without an API key
- Optional bring-your-own API keys for access to a wide range of third-party models
- Optional real-time voice chat (requires xAI API key)

LLM Pro:
- Free cloud AI model available without a subscription
- Premium subscription unlocks curated high-performance cloud AI models
- Vision-capable models for image analysis
- Optional bring-your-own API keys for access to additional third-party models

What is AI image generation in LLM Pro?

LLM Pro includes built-in AI image generation and image-to-image editing. The premium subscription includes up to 100 images per month. Image usage is clearly displayed in-app.

What is real-time web search?

Real-time web search allows supported AI models to retrieve up-to-date information. Availability depends on the selected model and requires an internet connection.

What is real-time Voice Chat?

Real-time Voice Chat enables spoken conversations with AI using supported providers. This feature requires your own compatible API key (such as xAI). Voice data is processed by the selected provider.

How is my data handled?

Free Cloud AI: Messages are sent to the AI provider to generate responses. Conversations are not stored on our servers.

LLM Pro – Premium Features: Messages and image prompts are processed via supported AI providers. We do not store conversations, images, or documents.

Bring Your Own API Key: Content is sent directly to the selected provider. API keys are stored only on your device.

Voice Chat: Voice data is sent directly to the voice provider. No voice data or transcripts are stored on our servers.

Can I use the app offline?

Chat LLM: Requires an internet connection for cloud features.

LLM Pro: Requires an internet connection for all features.

Why can't I upload images or documents?

Image and document uploads are only available when using models that support vision or document input. Feature availability depends on the selected model.

What is the LLM Pro subscription?

LLM Pro offers a £9.99/month subscription that unlocks premium cloud AI models, real-time web search on supported models, and AI image generation with a monthly limit.

What devices are supported?

Both apps support iPhone and iPad running iOS 18.0 or later.
