1. Description of the Services

Chat LLM provides multiple ways to interact with AI:

Built-in Local Model (Optional): Chat LLM includes an optional local AI model that runs on your device. When using a local model, no internet connection is required and no data is sent to external servers.

Free Cloud AI: Chat LLM provides a free cloud AI option that does not require an API key. This service requires an internet connection and may be subject to shared usage limits across users. Free cloud AI is text-only and does not support image or document uploads.

Voice Chat (Optional): Chat LLM may support real-time voice conversations using third-party providers. This feature requires you to provide your own compatible API key (for example, xAI). You are responsible for any API usage costs charged by the provider.

Bring Your Own API Key (Optional): Chat LLM may allow you to connect your own API keys to use third-party AI services such as OpenRouter, OpenAI, or xAI. When using this option, the App sends your content (text, and where supported images/documents/voice) to the selected provider and displays the responses returned. We do not operate or control third-party services or models.

LLM Pro provides the following AI interaction options:

Free Cloud AI: LLM Pro includes a free cloud AI option that does not require an API key. This service requires an internet connection and may be subject to shared usage limits across users. Free cloud AI is text-only and does not support image or document uploads.

Premium Subscription: LLM Pro offers a premium subscription service (£9.99 per month) that unlocks:

  • Access to curated high-performance cloud AI models
  • Real-time web search on supported models
  • Vision capabilities on supported models (for image analysis)
  • AI image generation and image-to-image editing with a monthly allowance

AI Image Generation (Included with Subscription): LLM Pro includes image generation and editing provided via third-party services (including Replicate). Image generation requires an internet connection. A monthly image allowance applies and is displayed in-app (currently up to 100 images per month).

Thinking Box (Optional): LLM Pro may display intermediate reasoning for supported models. Availability depends on the selected model/provider and may vary.

Voice Chat (Optional): LLM Pro may support real-time voice conversations using third-party providers. This feature requires you to provide your own compatible API key (for example, xAI). You are responsible for any API usage costs charged by the provider.

Bring Your Own API Key (Optional): LLM Pro may allow you to connect your own API keys to use third-party services such as OpenRouter, OpenAI, or xAI. When using this option, the App sends your content (text, and where supported images/documents/voice) to the selected provider and displays the responses returned. We do not operate or control third-party services or models.
