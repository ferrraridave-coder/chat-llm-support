1. Description of the Service

Chat LLM provides four ways to interact with AI:

Built-in Local Model: The App includes a built-in local AI model (Llama 3.2 1B) that runs entirely on your device and works completely offline. No account, API key, or internet connection is required to use the local model.

Cloud Inference: The App offers a free cloud-based AI option (Llama 3.1 8B) that does not require an API key. This service is subject to shared usage limits across all users and requires an internet connection. Cloud inference is text-only and does not support image or document uploads.

Voice Chat: The App offers real-time voice conversation powered by Grok's speech API. This feature requires you to provide your own xAI API key obtained from x.ai. Voice chat provides unlimited continuous responses without time limits and automatically saves conversation transcripts to your device. You are responsible for any API usage costs charged by xAI.

Bring Your Own API Key: The App optionally allows you to connect your own API keys to interact with additional AI models provided by third-party services such as OpenRouter, OpenAI, and xAI. When using cloud models with your own API keys, the App sends your text, images, documents, or voice input to these providers' APIs and displays the responses returned. We do not operate or control the third-party models.
